# 商品情報

## 事業内容

一般企業向けに、AIを使った業務フロー改善システムを導入する。

## 主なサービス

- 業務ヒアリング
- AI活用ポイントの抽出
- ChatGPT / API / 自動化ツールの導入設計
- 社内マニュアル作成
- 運用改善
- 月額保守

## 初期商品

### AI業務フロー診断

- 価格：50,000円
- 標準作業時間：8時間
- 内容：現状業務を整理し、AI化できる箇所を提案する

### AIフロー導入パッケージ

- 価格：300,000円
- 標準作業時間：40時間
- 内容：ヒアリング、設計、プロンプト作成、簡易自動化を行う

### 月額改善サポート

- 価格：50,000円/月
- 内容：改善相談、プロンプト修正、運用支援を行う
